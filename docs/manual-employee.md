## 1. การเข้าสู่ระบบ

1. เปิดเว็บไซต์ระบบ Check In/Out
2. ใส่ **รหัสพนักงาน** (เช่น EMP001)
3. ใส่ **รหัสผ่าน**
4. กดปุ่ม **"เข้าสู่ระบบ"**

> หากลืมรหัสผ่าน ให้ติดต่อผู้ดูแลระบบเพื่อ Reset

---

## 2. การติดตั้งแอป

เมื่อเปิดเว็บครั้งแรก จะมีหน้าแนะนำให้ติดตั้งแอป:

**Android (Chrome):**
- กดปุ่ม **"📲 ติดตั้งเลย"**
- หรือ กดเมนู ⋮ → "Install app"

**iPhone (Safari):**
- กดปุ่ม **⬆️ Share** ด้านล่าง
- เลือก **"Add to Home Screen"**
- กด **"Add"**

หลังติดตั้ง จะมีไอคอนแอปบนหน้าจอหลัก เปิดใช้งานได้เลยโดยไม่ต้องเปิด Browser

---

## 3. Check In (ลงเวลาเข้างาน)

1. ไปที่เมนู **"⏰ Check In/Out"**
2. กดปุ่ม **"📷 Check In"**
3. **ถ่ายรูป** — กล้องจะเปิดขึ้น ถ่ายรูปตัวเอง แล้วกด "✅ ใช้รูปนี้"
4. **ยืนยันตำแหน่ง** — แผนที่จะแสดงตำแหน่งปัจจุบัน
   - ถ้าตำแหน่งถูกต้อง → กด **"✅ ยืนยัน Check In"**
   - ถ้าตำแหน่งไม่ตรง → กด **"📍 ตำแหน่งไม่ตรง? กดที่นี่เพื่อหาตำแหน่งใหม่"**
5. ระบบจะบันทึกเวลา + รูปถ่าย + ตำแหน่ง GPS

> **สำคัญ:** ต้องเปิด GPS/Location บนมือถือก่อน Check In

---

## 4. Check Out (ลงเวลาออกงาน)

1. ไปที่เมนู **"⏰ Check In/Out"**
2. กดปุ่ม **"📍 Check Out"**
3. **ยืนยันตำแหน่ง** — แผนที่จะแสดงตำแหน่งปัจจุบัน
4. กด **"✅ ยืนยัน Check Out"**

> Check Out ไม่ต้องถ่ายรูป แค่ยืนยันตำแหน่ง GPS

---

## 5. การขอลา

### ขอลาใหม่:
1. ไปที่เมนู **"📋 ขอลา"**
2. กดปุ่ม **"+ ขอลาใหม่"**
3. กรอกข้อมูล:
   - ประเภทการลา (ลาป่วย, ลากิจ, ลาพักร้อน ฯลฯ)
   - วันที่เริ่ม - วันที่สิ้นสุด
   - เหตุผล
4. กด **"ส่งคำขอลา"**
5. ระบบจะส่งคำขอไปยังหัวหน้าแผนกอัตโนมัติ

### ดูสถานะการลา:
- **สีเหลือง** = รออนุมัติ
- **สีเขียว** = อนุมัติแล้ว
- **สีแดง** = ไม่อนุมัติ

---

## 6. ดูประกาศ

1. ไปที่เมนู **"📢 ประกาศ"**
2. คลิกที่ประกาศเพื่ออ่านรายละเอียด
3. ถ้ามีไฟล์แนบ สามารถกดดาวน์โหลดได้

---

## 7. ดูปฏิทิน

1. ไปที่เมนู **"📅 ปฏิทิน"**
2. เลื่อนเดือนด้วยปุ่ม ◀ ▶
3. สีของวัน:
   - 🟢 จุดเขียว = มาทำงาน
   - 🟡 จุดเหลือง = ลา
   - 🔴 จุดแดง = วันหยุดราชการ
   - ⚪ จุดเทา = เสาร์-อาทิตย์
4. คลิกที่วันเพื่อดูรายละเอียด

---

## 8. ดูรายงาน

1. ไปที่เมนู **"📊 รายงาน"**
2. เลือกประเภทรายงาน (การเข้างาน / การลา / สรุปรายเดือน)
3. เลือกช่วงวันที่
4. กด **"🔍 ดูรายงาน"**

> พนักงานจะเห็นเฉพาะข้อมูลของตัวเองเท่านั้น

---

## 9. เปลี่ยนรหัสผ่าน

1. ไปที่เมนู **"🔑 เปลี่ยนรหัสผ่าน"** (อยู่ในเมนู "⋯ เพิ่มเติม" บนมือถือ)
2. ใส่รหัสผ่านปัจจุบัน
3. ใส่รหัสผ่านใหม่ (อย่างน้อย 4 ตัวอักษร)
4. ใส่รหัสผ่านใหม่อีกครั้งเพื่อยืนยัน
5. กด **"🔑 เปลี่ยนรหัสผ่าน"**

---

## 10. เปลี่ยนรูปโปรไฟล์

1. ไปที่ **หน้าหลัก (Dashboard)**
2. คลิกที่ **รูปวงกลม** ข้างชื่อ (มีไอคอน 📷 เล็กๆ)
3. เลือกรูปจากเครื่อง
4. รูปจะอัปโหลดอัตโนมัติ

---

## 11. การแจ้งเตือน

- กดไอคอน **🔔 กระดิ่ง** ที่ Header
- จะแสดงรายการแจ้งเตือน เช่น:
  - ✅ การลาได้รับอนุมัติ
  - ❌ การลาไม่ได้รับอนุมัติ
  - 📋 ส่งคำขอลาสำเร็จ
- กดที่แจ้งเตือนเพื่อไปหน้าที่เกี่ยวข้อง

---

## 12. ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|---|---|
| GPS ไม่ตรง | เปิด GPS ในตั้งค่า → ตั้ง Location Mode = High Accuracy → เปิด Precise Location |
| กล้องไม่เปิด | ต้องใช้ HTTPS + อนุญาต Camera ใน Browser |
| ลืมรหัสผ่าน | ติดต่อผู้ดูแลระบบให้ Reset |
| Check In ไม่ได้ | ตรวจสอบว่ายังไม่ได้ Check In วันนี้ |
| แอปโหลดช้า | ลอง Hard Refresh (Ctrl+Shift+R) |
