# คู่มือติดตั้งแอป Check In/Out ลงมือถือ/คอม

## รองรับทุก Browser: Chrome, Edge, Safari

---

## Android — Microsoft Edge

### วิธีที่ 1: กดจาก popup
1. เปิด Edge → ไปที่เว็บระบบ Check In/Out
2. จะมี popup ขึ้นมา **"ติดตั้งแอป"**
3. กด **"📲 ติดตั้งเลย"**
4. เสร็จ! ไอคอนจะอยู่บนหน้าจอหลัก

### วิธีที่ 2: จากเมนู
1. เปิด Edge → ไปที่เว็บระบบ
2. กดเมนู **"⋯"** (จุดสามจุดด้านล่าง)
3. เลื่อนหา **"Add to phone"** หรือ **"เพิ่มไปยังโทรศัพท์"**
4. กด **"Add"** หรือ **"เพิ่ม"**
5. เสร็จ!

---

## Android — Google Chrome

### วิธีที่ 1: กดจาก popup
1. เปิด Chrome → ไปที่เว็บระบบ
2. จะมี popup **"ติดตั้งแอป"**
3. กด **"📲 ติดตั้งเลย"**

### วิธีที่ 2: จากเมนู
1. กดเมนู **"⋮"** (จุดสามจุดมุมขวาบน)
2. กด **"Install app"** หรือ **"ติดตั้งแอป"**
3. กด **"Install"** หรือ **"ติดตั้ง"**

---

## iPhone — Microsoft Edge

1. เปิด Edge → ไปที่เว็บระบบ
2. กดเมนู **"⋯"** (จุดสามจุดด้านล่างตรงกลาง)
3. เลื่อนหา **"Add to Home Screen"** หรือ **"เพิ่มไปยังหน้าจอหลัก"**
4. กด **"Add"** หรือ **"เพิ่ม"**
5. เสร็จ!

> ⚠️ ถ้าไม่เห็น "Add to Home Screen" ใน Edge → ใช้ Safari แทน (iOS รองรับ PWA เต็มที่เฉพาะ Safari)

---

## iPhone — Safari (แนะนำสำหรับ iOS)

1. เปิด **Safari** → ไปที่เว็บระบบ
2. กดปุ่ม **⬆️ Share** (ไอคอนสี่เหลี่ยมมีลูกศรชี้ขึ้น ด้านล่าง)
3. เลื่อนหา **"Add to Home Screen"** หรือ **"เพิ่มไปยังหน้าจอหลัก"**
4. แก้ชื่อ (ถ้าต้องการ) → กด **"Add"** หรือ **"เพิ่ม"**
5. เสร็จ! ไอคอนจะอยู่บนหน้าจอหลัก

---

## คอมพิวเตอร์ — Microsoft Edge

1. เปิด Edge → ไปที่เว็บระบบ
2. จะเห็นไอคอน **"+"** หรือ **"🖥️"** ที่ Address bar (ด้านขวา)
3. กด → กด **"Install"** หรือ **"ติดตั้ง"**
4. แอปจะเปิดเป็นหน้าต่างแยก (ไม่มี address bar)

### หรือ:
1. กดเมนู **"⋯"** (มุมขวาบน)
2. ไปที่ **"Apps"** → **"Install this site as an app"**
3. กด **"Install"**

---

## คอมพิวเตอร์ — Google Chrome

1. เปิด Chrome → ไปที่เว็บระบบ
2. กดไอคอน **"⬇️"** ที่ Address bar (ด้านขวา)
3. กด **"Install"**

### หรือ:
1. กดเมนู **"⋮"** (มุมขวาบน)
2. กด **"Install Check In/Out..."** หรือ **"ติดตั้ง..."**

---

## หลังติดตั้งแล้ว

- ✅ มีไอคอนบนหน้าจอหลัก (เหมือนแอปปกติ)
- ✅ เปิดแล้วไม่มี address bar (เต็มจอ)
- ✅ ไม่ต้องพิมพ์ URL ทุกครั้ง
- ✅ เปิดได้เร็วกว่าเปิดผ่าน Browser

---

## ถอนการติดตั้ง

**Android:**
- กดค้างที่ไอคอนแอป → "Uninstall" หรือ "ถอนการติดตั้ง"

**iPhone:**
- กดค้างที่ไอคอน → "Remove App" → "Delete from Home Screen"

**คอม (Edge/Chrome):**
- เปิดแอป → กดเมนู "⋯" → "Uninstall"

---

## ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|---|---|
| ไม่เห็น popup ติดตั้ง | ใช้วิธีที่ 2 (จากเมนู) |
| iPhone Edge ไม่มี Add to Home Screen | ใช้ Safari แทน |
| ติดตั้งแล้วแต่เปิดไม่ได้ | ลองถอนแล้วติดตั้งใหม่ |
| ไอคอนไม่สวย | ปกติ ขึ้นกับ Browser/OS |
