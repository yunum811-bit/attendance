## 1. ภาพรวมสิทธิ์ตาม Role

| Role | สิทธิ์ |
|---|---|
| **employee** (พนักงาน) | Check In/Out, ขอลา, ดูรายงานตัวเอง |
| **manager** (หัวหน้าแผนก) | + อนุมัติลาพนักงานในแผนก, ดูรายงานแผนก |
| **md** (กรรมการผู้จัดการ) | + อนุมัติลาหัวหน้าแผนก, ดูรายงานทั้งหมด |
| **admin** (ผู้ดูแลระบบ) | + จัดการพนักงาน, ตั้งค่าระบบ, ยกเลิกอนุมัติ, ทุกอย่าง |
| **manager_admin** (หัวหน้า+Admin) | สิทธิ์ Manager + Admin รวมกัน |

## 2. จัดการพนักงาน

เมนู: **"👥 พนักงาน"**

### เพิ่มพนักงานใหม่:
1. กด **"+ เพิ่มพนักงาน"**
2. กรอก: รหัสพนักงาน, ชื่อ, นามสกุล, อีเมล, แผนก, บทบาท, รหัสผ่าน
3. กด **"บันทึก"**

> รหัสพนักงานต้องไม่ซ้ำ และ **แก้ไขไม่ได้** ภายหลัง

### แก้ไขข้อมูลพนักงาน:
1. กดปุ่ม **"✏️ แก้ไข"** ที่แถวพนักงาน
2. แก้ไขได้: ชื่อ, นามสกุล, อีเมล, แผนก, บทบาท
3. แก้ไขไม่ได้: รหัสพนักงาน
4. กด **"💾 บันทึก"**

### จัดการรหัสผ่าน:
- **"🔑 รหัสผ่าน"** — ตั้งรหัสผ่านใหม่ให้พนักงาน
- **"🔄 Reset"** — รีเซ็ตเป็น "1234"

### ลบพนักงาน:
- กดปุ่ม **"🗑️ ลบ"** ที่แถวพนักงาน
- ลบได้เฉพาะพนักงานที่ **ไม่มีประวัติ** Check In/ลา
- ถ้ามีประวัติอยู่ → ลบไม่ได้ (ให้เปลี่ยน role หรือย้ายแผนกแทน)

## 3. อนุมัติการลา

เมนู: **"✅ อนุมัติ"**

### ช่อง "คำขอลาจากพนักงาน":
- Admin เห็นคำขอจากพนักงาน **ทุกแผนก**
- รวมพนักงานที่ไม่มีหัวหน้าแผนก
- กด **"✅ อนุมัติ"** หรือ **"❌ ปฏิเสธ"** (ต้องระบุเหตุผล)

### ช่อง "คำขอลาจากหัวหน้าแผนก" (MD/Admin Only):
- แสดงคำขอจากหัวหน้าแผนกทุกแผนก
- เฉพาะ MD หรือ Admin อนุมัติได้

---

## 4. ยกเลิกการอนุมัติ

เมนู: **"✅ อนุมัติ"** → ส่วน **"อนุมัติแล้ว (ยกเลิกได้)"**

### วิธีใช้:
1. เลื่อนลงดูส่วน "อนุมัติแล้ว"
2. กดปุ่ม **"⚠️ ยกเลิกอนุมัติ"**
3. ระบุเหตุผล (หรือเว้นว่าง)
4. ระบบจะ:
   - เปลี่ยนสถานะเป็น **"ยกเลิกการอนุมัติ"** (สีส้ม)
   - **แจ้งเตือนพนักงานทันที** ว่าการลาถูกยกเลิก พร้อมเหตุผล
   - วันลาจะ **ไม่ถูกนับ** ในโควต้าอีกต่อไป

> เฉพาะ Admin / manager_admin เท่านั้นที่ยกเลิกได้

## 5. อนุมัติ "ลืม Check In"

เมนู: **"✅ อนุมัติ"** → ส่วน **"⏰ ลืม Check In — รออนุมัติ"**

เมื่อพนักงานลืม Check In แล้วส่งคำขอบันทึกเวลาย้อนหลัง:

1. ตรวจสอบข้อมูล: วันที่, เวลาเข้า/ออก, เหตุผล
2. กด **"✅ อนุมัติ"** → ระบบจะบันทึกเวลาเข้าประวัติให้อัตโนมัติ
3. หรือกด **"❌ ปฏิเสธ"**

> เมื่ออนุมัติแล้ว เวลาจะปรากฏในประวัติ Check In/Out ของพนักงาน + รายงาน

## 6. จัดการประเภทการลา

เมนู: **"📝 ประเภทการลา"**

### เพิ่มประเภทใหม่:
1. กด **"+ เพิ่มประเภทการลา"**
2. กรอก:
   - ชื่อ (เช่น "ลาบวช")
   - จำนวนวันสูงสุด/ปี (ใส่ **0** = กำหนดรายบุคคลในหน้า "กำหนดวันลา")
   - คำอธิบาย
3. กด **"💾 เพิ่มประเภทการลา"**

### แก้ไข/ลบ:
- **"✏️ แก้ไข"** — เปลี่ยนชื่อ, จำนวนวัน
- **"🗑️ ลบ"** — ลบได้เฉพาะประเภทที่ไม่มีคนใช้

---

## 6. กำหนดวันลารายบุคคล

เมนู: **"📅 กำหนดวันลา"**

กำหนดวันลาเฉพาะบุคคล (ไม่เท่ากัน):

1. เลือกปี
2. กดปุ่ม **"✏️ แก้ไข"** ที่พนักงานที่ต้องการ
3. ใส่จำนวนวันลาแต่ละประเภท
4. ถ้าเว้นว่าง = ใช้ค่าเริ่มต้นของระบบ
5. กด **"💾 บันทึก"**

---

## 7. สร้างประกาศ

เมนู: **"📢 ประกาศ"**

### สร้างประกาศใหม่:
1. กด **"+ สร้างประกาศ"**
2. กรอก: หัวข้อ, เนื้อหา
3. แนบไฟล์ (ถ้ามี) — รองรับ PDF, Word, Excel, รูปภาพ (สูงสุด 5MB)
4. กด **"📢 เผยแพร่ประกาศ"**

### ลบประกาศ:
- กดไอคอน **"🗑️"** ที่มุมขวาของประกาศ

## 8. จัดการปฏิทิน/วันหยุด

เมนู: **"📅 ปฏิทิน"**

### วันหยุดในระบบมี 2 ประเภท:

| ประเภท | ตัวอย่าง | แสดงอัตโนมัติ |
|---|---|---|
| **วันหยุดคงที่** | วันขึ้นปีใหม่, วันจักรี, สงกรานต์, วันแม่, วันพ่อ ฯลฯ | ✅ ทุกปี (ไม่ต้องเพิ่มเอง) |
| **วันหยุดที่เปลี่ยนทุกปี** | มาฆบูชา, วิสาขบูชา, อาสาฬหบูชา, เข้าพรรษา, วันชดเชย | ❌ Admin ต้องเพิ่มเองทุกต้นปี |

### เพิ่มวันหยุด:
1. เลื่อนไปเดือนที่ต้องการ
2. กด **"+ เพิ่มวันหยุด"**
3. เลือกวันที่ + ใส่ชื่อวันหยุด
4. กด **"💾 เพิ่มวันหยุด"**

### แก้ไขวันหยุดที่ไม่ตรง:
- กด **✏️** ที่วันหยุดที่ต้องการแก้ไข (ทั้งวันหยุดคงที่และวันหยุดที่เพิ่มเอง)
- แก้ไขชื่อ หรือ เปลี่ยนวันที่
- กด **"💾 บันทึก"**

> ตัวอย่าง: วันจักรี (6 เม.ย.) ตรงวันเสาร์ → กด ✏️ → เพิ่ม "วันชดเชยวันจักรี" วันที่ 8 เม.ย.

### ลบวันหยุด:
- กด **🗑️** ที่วันหยุดที่เพิ่มเอง

### สิ่งที่ Admin ต้องทำทุกต้นปี:
1. ตรวจสอบวันหยุดทางพุทธศาสนา (มาฆบูชา, วิสาขบูชา, อาสาฬหบูชา, เข้าพรรษา)
2. เพิ่มวันหยุดเหล่านี้ + วันชดเชย (ประมาณ 5-7 วัน)
3. ตรวจว่าวันหยุดคงที่ตรงกับวันหยุดจริงหรือไม่ (ถ้าตรงเสาร์-อาทิตย์ ต้องเพิ่มวันชดเชย)

## 9. ดูรายงาน + Export

เมนู: **"📊 รายงาน"**

### ประเภทรายงาน:
1. **รายงานการเข้างาน** — วันที่, เวลาเข้า/ออก, ตำแหน่ง GPS, มาสาย
2. **รายงานการลา** — ประเภท, วันที่, สถานะ, ผู้อนุมัติ
3. **สรุปรายเดือน** — จำนวนวันมาทำงาน, มาสาย, ลา

### สิทธิ์:
- Admin เห็น **ทุกแผนก** + เลือกกรองตามแผนกได้
- Manager เห็นเฉพาะแผนกตัวเอง
- พนักงานเห็นเฉพาะตัวเอง

### Export:
- กด **"📄 Export CSV"** — เปิดใน Excel ได้
- กด **"📊 Export XLSX"** — ไฟล์ Excel โดยตรง

## 10. ตั้งค่าบริษัท

เมนู: **"⚙️ ตั้งค่า"**

### ชื่อบริษัท:
- พิมพ์ชื่อ → กด "💾 บันทึกชื่อ"
- แสดงที่ Header + หน้า Login

### โลโก้บริษัท:
- เลือกรูป (PNG, JPG, GIF, SVG สูงสุด 2MB)
- กด "📤 อัปโหลดโลโก้"
- แสดงที่ Header + หน้า Login

### เวลาทำงาน:
- ตั้งเวลาเข้างาน (08:30)
- ตั้งเวลาเลิกงาน (17:30)
- ตั้งเวลาพักเที่ยง (12:00 - 13:00)
- กด "💾 บันทึกเวลาทำงาน"

> เวลาทำงานจะแสดงที่ Dashboard และหน้า Check In/Out
> Check In หลังเวลาเข้างาน = แสดงสีแดง "(สาย)"

## 11. จัดการรหัสผ่าน

### ตั้งรหัสผ่านให้พนักงาน:
1. ไปที่ "👥 พนักงาน"
2. กด **"🔑 รหัสผ่าน"** → ใส่รหัสใหม่ → บันทึก

### Reset เป็นค่าเริ่มต้น:
- กด **"🔄 Reset"** → รหัสผ่านจะเป็น "1234"

## 12. ลำดับการอนุมัติ

```
┌─────────────────────────────────────────────────┐
│  พนักงาน ขอลา                                   │
│     ↓                                           │
│  มีหัวหน้าแผนก? ─── ใช่ → หัวหน้าแผนก อนุมัติ  │
│     │                                           │
│     └── ไม่มี → Admin/MD อนุมัติ                 │
├─────────────────────────────────────────────────┤
│  หัวหน้าแผนก ขอลา → MD/Admin อนุมัติ            │
├─────────────────────────────────────────────────┤
│  Admin ยกเลิกอนุมัติ → แจ้งพนักงานทันที          │
└─────────────────────────────────────────────────┘
```
