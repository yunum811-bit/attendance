## 1. การเข้าสู่ระบบ

1. เปิดเว็บไซต์ระบบ Check In/Out
2. ใส่ **รหัสพนักงาน** (เช่น EMP001)
3. ใส่ **รหัสผ่าน**
4. กดปุ่ม **"เข้าสู่ระบบ"**

> หากลืมรหัสผ่าน ให้ติดต่อผู้ดูแลระบบเพื่อ Reset

---

## 2. การติดตั้งแอป (Microsoft Edge)

### Android — ติดตั้งผ่าน Edge:

1. เปิดแอป **Microsoft Edge** บนมือถือ
2. พิมพ์ URL ของระบบ Check In/Out ใน address bar
3. รอหน้าเว็บโหลดเสร็จ
4. **วิธี A:** ถ้ามี popup "ติดตั้งแอป" ขึ้นมา → กด **"📲 ติดตั้งเลย"**
5. **วิธี B:** ถ้าไม่มี popup:
   - กดเมนู **"⋯"** (จุดสามจุด ด้านล่างตรงกลาง)
   - เลื่อนหา **"Add to phone"** หรือ **"เพิ่มไปยังโทรศัพท์"**
   - กด **"Add"** หรือ **"เพิ่ม"**
6. ไอคอนแอปจะปรากฏบนหน้าจอหลัก ✅

### iPhone — ติดตั้งผ่าน Edge:

1. เปิดแอป **Microsoft Edge** บน iPhone
2. พิมพ์ URL ของระบบ Check In/Out ใน address bar
3. รอหน้าเว็บโหลดเสร็จ
4. กดเมนู **"⋯"** (จุดสามจุด ด้านล่างตรงกลาง)
5. เลื่อนหา **"Add to Home Screen"** หรือ **"เพิ่มไปยังหน้าจอหลัก"**
6. กด **"Add"** หรือ **"เพิ่ม"**
7. ไอคอนแอปจะปรากฏบนหน้าจอหลัก ✅

> **หมายเหตุ iPhone:** ถ้าไม่เห็น "Add to Home Screen" ใน Edge ให้ใช้ **Safari** แทน:
> Safari → กดปุ่ม ⬆️ Share (ด้านล่าง) → "Add to Home Screen" → "Add"

### หลังติดตั้งแล้ว:
- ✅ มีไอคอนบนหน้าจอหลัก (เหมือนแอปปกติ)
- ✅ เปิดแล้วไม่มี address bar (เต็มจอ)
- ✅ ไม่ต้องพิมพ์ URL ทุกครั้ง
- ✅ เปิดได้เร็วกว่าเปิดผ่าน Browser

## 3. Check In (ลงเวลาเข้างาน)

**เวลาทำงาน: 08:30 - 17:30 น.**

1. ไปที่เมนู **"⏰ Check In/Out"**
2. กดปุ่ม **"📷 Check In"**
3. **ถ่ายรูป** — กล้องจะเปิดขึ้น ถ่ายรูปตัวเอง แล้วกด "✅ ใช้รูปนี้"
4. **ยืนยันตำแหน่ง** — แผนที่จะแสดงตำแหน่งปัจจุบัน
   - ถ้าตำแหน่งถูกต้อง → กด **"✅ ยืนยัน Check In"**
   - ถ้าตำแหน่งไม่ตรง → กด **"📍 ตำแหน่งไม่ตรง? กดที่นี่เพื่อหาตำแหน่งใหม่"**
5. ระบบจะบันทึกเวลา + รูปถ่าย + ตำแหน่ง GPS

> **สำคัญ:** ต้องเปิด GPS/Location บนมือถือก่อน Check In
> **หมายเหตุ:** ถ้า Check In หลัง 08:30 น. เวลาจะแสดงเป็น **สีแดง** พร้อมข้อความ "(สาย)"

---

## 4. Check Out (ลงเวลาออกงาน)

1. ไปที่เมนู **"⏰ Check In/Out"**
2. กดปุ่ม **"📍 Check Out"**
3. **ยืนยันตำแหน่ง** — แผนที่จะแสดงตำแหน่งปัจจุบัน
4. กด **"✅ ยืนยัน Check Out"**

> Check Out **ไม่ต้องถ่ายรูป** แค่ยืนยันตำแหน่ง GPS

---

## 5. ลืม Check In (ขอบันทึกเวลาย้อนหลัง)

ถ้าลืม Check In หรือมือถือมีปัญหา สามารถขอบันทึกเวลาย้อนหลังได้:

1. ไปที่เมนู **"⏰ Check In/Out"**
2. กดลิงก์ **"⏰ ลืม Check In? กดที่นี่"** (ใต้ปุ่ม Check In/Out)
3. กรอกข้อมูล:
   - **วันที่** — วันที่ลืม Check In
   - **เวลาเข้างาน** — เวลาที่มาถึงจริง
   - **เวลาออกงาน** — (ถ้ามี)
   - **เหตุผล** — เช่น "ลืมกดเช็คอิน", "มือถือแบตหมด"
4. กด **"📨 ส่งคำขอ"**
5. ระบบจะส่งคำขอไปยังหัวหน้าแผนก
6. เมื่อหัวหน้าอนุมัติ → ระบบจะบันทึกเวลาเข้าประวัติให้อัตโนมัติ

> **หมายเหตุ:** ต้องรอหัวหน้าแผนกอนุมัติก่อน เวลาถึงจะถูกบันทึก

## 6. การขอลา

### รูปแบบการลา:
| รูปแบบ | รายละเอียด |
|---|---|
| **ลาเต็มวัน** | เลือกวันที่เริ่ม - สิ้นสุด |
| **ลาครึ่งวัน** | เลือกช่วงเช้า (08:30-12:00) หรือบ่าย (13:00-17:30) |
| **ลารายชั่วโมง** | เลือกเวลาเริ่ม-สิ้นสุด (ทุก 30 นาที) |

### ขอลาใหม่:
1. ไปที่เมนู **"📋 ขอลา"**
2. กดปุ่ม **"+ ขอลาใหม่"**
3. กรอกข้อมูล:
   - ประเภทการลา (ลาป่วย, ลากิจ, ลาพักร้อน ฯลฯ)
   - รูปแบบการลา (เต็มวัน / ครึ่งวัน / รายชั่วโมง)
   - วันที่ / เวลา
   - เหตุผล
4. กด **"ส่งคำขอลา"**
5. ระบบจะส่งคำขอไปยังหัวหน้าแผนกอัตโนมัติ (ถ้าไม่มีหัวหน้าจะส่งไป Admin)

### สถานะการลา:
| สี | สถานะ | ความหมาย |
|---|---|---|
| 🟡 เหลือง | รออนุมัติ | ส่งแล้ว รอหัวหน้าอนุมัติ |
| 🟢 เขียว | อนุมัติแล้ว | ลาได้ |
| 🔴 แดง | ไม่อนุมัติ | ถูกปฏิเสธ |
| 🟠 ส้ม | ยกเลิกการอนุมัติ | Admin ยกเลิกหลังอนุมัติแล้ว |

## 7. ดูประกาศ

1. ไปที่เมนู **"📢 ประกาศ"**
2. คลิกที่ประกาศเพื่ออ่านรายละเอียด
3. ถ้ามีไฟล์แนบ สามารถกดดาวน์โหลดได้

---

## 8. ดูปฏิทิน

1. ไปที่เมนู **"📅 ปฏิทิน"**
2. เลื่อนเดือนด้วยปุ่ม ◀ ▶
3. สีของวัน:
   - 🟢 จุดเขียว = มาทำงาน
   - 🟡 จุดเหลือง = ลา
   - 🔴 จุดแดง = วันหยุดราชการ / วันหยุดบริษัท
   - ⚪ จุดเทา = เสาร์-อาทิตย์
4. คลิกที่วันเพื่อดูรายละเอียด
5. ด้านล่างแสดงวันหยุดของเดือนนั้น

---

## 9. ดูรายงาน

1. ไปที่เมนู **"📊 รายงาน"**
2. เลือกประเภทรายงาน (การเข้างาน / การลา / สรุปรายเดือน)
3. เลือกช่วงวันที่
4. กด **"🔍 ดูรายงาน"**

> พนักงานจะเห็นเฉพาะข้อมูลของตัวเองเท่านั้น

---

## 10. เปลี่ยนรหัสผ่าน

1. ไปที่เมนู **"🔑 เปลี่ยนรหัสผ่าน"** (อยู่ในเมนู "⋯ เพิ่มเติม" บนมือถือ)
2. ใส่รหัสผ่านปัจจุบัน
3. ใส่รหัสผ่านใหม่ (อย่างน้อย 4 ตัวอักษร)
4. ใส่รหัสผ่านใหม่อีกครั้งเพื่อยืนยัน
5. กด **"🔑 เปลี่ยนรหัสผ่าน"**

---

## 11. เปลี่ยนรูปโปรไฟล์

1. ไปที่ **หน้าหลัก (Dashboard)**
2. คลิกที่ **รูปวงกลม** ข้างชื่อ (มีไอคอน ✏️)
3. เลือกรูปจากเครื่อง (PNG, JPG, GIF สูงสุด 2MB)
4. รูปจะอัปโหลดอัตโนมัติ

---

## 12. การแจ้งเตือน

- กดไอคอน **🔔 กระดิ่ง** ที่ Header
- แจ้งเตือนที่จะได้รับ:
  - 📋 ส่งคำขอลาสำเร็จ (บอกว่าส่งไปหาใคร)
  - ✅ การลาได้รับอนุมัติ
  - ❌ การลาไม่ได้รับอนุมัติ
  - ⚠️ การลาถูกยกเลิก (Admin ยกเลิกหลังอนุมัติ)
- กดที่แจ้งเตือนเพื่อไปหน้าที่เกี่ยวข้อง

---

## 13. ปัญหาที่พบบ่อย

| ปัญหา | วิธีแก้ |
|---|---|
| GPS ไม่ตรง | เปิด GPS ในตั้งค่า → ตั้ง Location Mode = High Accuracy → เปิด Precise Location |
| กล้องไม่เปิด | ต้องใช้ HTTPS + อนุญาต Camera ใน Browser |
| ลืมรหัสผ่าน | ติดต่อผู้ดูแลระบบให้ Reset |
| Check In ไม่ได้ | ตรวจสอบว่ายังไม่ได้ Check In วันนี้ |
| แอปโหลดช้า | ลอง Hard Refresh (Ctrl+Shift+R) |
| GPS Samsung ไม่ตรง | Settings → Location → เปิด Google Location Accuracy + Precise Location |
